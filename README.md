# Android Compose Navigation Samples

A Kotlin Android coursework project demonstrating Jetpack Compose navigation, a splash screen, multiple activities, and small interactive mini-features including a pizza calculator and GPA-related functionality.

This repo is best positioned as an additional Android sample rather than a headline portfolio project. It is useful when the goal is to show learning progress in Compose navigation and multi-screen Android app structure.

# Project Summary

This project appears to be a class-based Android navigation exercise built with Kotlin and Jetpack Compose. The public repository structure shows:

* `MainActivity.kt`
* `MainActivity2.kt`
* `NavClass.kt`
* `PizzaPartyFun.kt`
* `gpaAppFun.kt`
* Compose UI theme files

The visible navigation code includes:

* a splash screen
* a first screen
* a second screen
* navigation between Compose destinations
* transition from Compose navigation into another activity
* a slider and checkbox interaction
* TODO markers for adding more navigation destinations

1. Open the project in Android Studio.
2. Sync Gradle dependencies.

5. Explore the additional activity and mini-feature screens.
